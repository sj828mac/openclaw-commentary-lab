# OpenClaw Commentary Lab

Internal testing blog for publishing OpenClaw-related tracking and commentary posts on GitHub Pages.

## Purpose

- Publish tracking/commentary posts quickly
- Compare multiple editorial styles on standalone pages
- Serve as a lightweight internal validation site before wider distribution

## Current post set

- `ai-hedge-fund` long-form commentary
- `ai-hedge-fund` short post
- `ai-hedge-fund` channel variants

## Local preview

Open `index.html` directly in a browser, or serve statically:

```bash
python3 -m http.server 8080
```

## Deployment

Published via GitHub Pages from the `main` branch root.
