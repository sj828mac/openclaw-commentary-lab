mempalace 短版快評 v1

`mempalace` 值得追，但真正值得看的，不是它記住了多少，而是它怎麼面對 AI 記憶系統最難的現實。

它最有意思的地方，在於不想太早做摘要裁切，而是主張保留原始內容，再靠 palace 結構與搜尋把東西找回來。這對 agent memory 來說是很強的主張。

但更關鍵的是：它現在已經進入公開壓力測試階段。

最近最值得看的不是口號，而是：

  • README 主動修正過度包裝的 benchmark / AAAK 敘事
  • hooks、MCP、search、plugin manifest 問題開始大量浮現
  • 假網站與安全治理問題也進場了

一句話：

mempalace 不是單純的記憶工具，而是一個正在被現實世界檢驗的 agent long-term memory 樣本。
